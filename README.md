# Игра Match-3 (C# & MonoGame)

Данный проект представляет собой реализацию игры в жанре Match-3, разработанную специально для вакансии программиста в Гейм Форест. Игра написана на языке C# с использованием фреймворка MonoGame, без применения готовых движков вроде Unity.

## 🏗️ Архитектура проекта

Проект построен с разделением логики, графики и управления (принципы чистого кода и State Machine):

### Core (Логика)
Математическая модель поля 8x8, алгоритмы поиска совпадений и обработки бонусов. Полностью отделена от рендеринга.

### Managers
Управление состояниями экранов (`ScreenManager`), обработка ввода мыши (`InputManager`) и эффекты времени.

### Screens
Отдельные классы для Главного меню, Игрового экрана и экрана Game Over.

### Animation System
Все перемещения реализованы через линейную интерполяцию (`Lerp`), что обеспечивает плавность 60 FPS.

## 🛠️ Стек технологий

- **Язык:** C#
- **Фреймворк:** MonoGame (DesktopGL)
- **Среда разработки:** Visual Studio 2022 / 2026
- **Управление ресурсами:** MonoGame Content Pipeline (MGCB) для компиляции шрифтов
